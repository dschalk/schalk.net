"JS-monads-part1" begins the Javascript Monads series in earnest. The previous publications were exploratory in nature.

"JS-monads-part2 demonstrates the handling of numeric and string data provided by a Haskell web server. The monads run the data through successive computations guided by user interaction with the browser. Most of the api is available for experimentation in the browser console.

Like Websockets Monads Part 2, Part 3 demonstrates guided iteration along a chain of monads, and also a tree with multiple branches. The centerpiece is "updateCalc", which controls the action in a dice game. The game is an expansion of the basic tree JS Monads Part 2. Quite a bit of functionality is handled with 31 lines of code. Click JS-monads-part3 for more information and a demonstration of the multi-player game.
